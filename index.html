CSS & Flexbox

CSS er brukt gjennomgående på hele siden for å style den. Bruken av CSS er relativt elementær noe grunnet utviklers litt begrensede kompetanse på området, men også at det ikke har vært et spesielt stort behov for noe mer. For å organisere bildene og dokumentasjonen er Flexbox benyttet. Dette ble valgt fremfor Grid fordi det automatiserte en del av arbeidet med å gjøre siden responsiv til ulike skjermstørrelser. For å sikre responsivitet er det brukt media-queries i CSSen for å tilpasse Flexboxene.

jQuery

På denne siden er jQuery brukt for å velge det klikkbare elementet som viser og skjuler dokumentasjonen. I dette tilfellet er bruken av jQuery svært begrenset, men ettersom det ikke var behov for noen mer kompleks og avansert bruk av biblioteket ble det valgt å holde det enkelt. De eneste funksjonene som ble brukt i forbindelse med jQuery var .click() og .toggleClass() for å manipulere accordion objektet til dokumentasjonen.

Testing

Utviklingen av siden er i hovedsak gjennomført på Chrome. Etter ferdigstilling er siden i tillegg blitt testet på både Safari og Firefox. Animasjonene og interaksjonen med de ser ut til å fungere som det skal, og det samme gjelder knappen for å vise og skjule dokumentasjonen. Tesingen kunne selvsagt vært automatisert og mer grundig, men utifra oppgavens krav så dette ut til å være tilstrekkelig.
